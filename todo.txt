MySQL++
=======

1. Download into source code into /usr/local/src
2. Followed README instructions to build of:

   $ mkdir build; cd build
   $ cmake ..
   $ make
   $ make install // sudo make install?

This adds the mysql++ headers in the /usr/include/mysql++, and it adds the dlls--actual and synmlinks--in /usr/lib/// sudo make install. both of which are system paths.
