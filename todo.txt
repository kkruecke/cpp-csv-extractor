MySQL Connector for C++ Notes
=============================

I built the MySQL Connector for C++, libmysqlcppconn7, and its development package, libmysqlcppconn-dev, from source by doing:

~ $ sudo mkdir /usr/local/src
~ $ chown -R root:kurt /usr/local/src

/usr/local/src $ sudo apt-get source --compile mysql-connector-c++

mysql++3 Alternative C++ Library for MySQL
==========================================

Pluses
------

stl-like in design and implementation. It also does have documentation and examples.

How to Install Directly
-----------------------

$ sudo aptitude install libmysql++-dev libmysql++3 libmysql++-doc

How to Build from Source to Also Get Examples
---------------------------------------------

$ sudo aptitude purge libmysql++-dev libmysql++3 libmysql++-doc

/usr/local/src $ sudo apt-get build-dep mysql++
/usr/local/src $ sudo apt-get source --compile mysql++

/usr/local/src $ /usr/local/src $ sudo dpkg -i libmysql++-devXXX libmysql++3XXX libmysql++-docXXX
